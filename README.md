# Rosco SPL - Sociedad Punta de Lobos

Plantilla lista con:
- Todas las letras A-Z con 3 preguntas por letra (Prevención de Riesgos).
- Logo de SPL integrado (arriba a la derecha).
- Cámara en el centro del rosco (toggle ON/OFF).
- Sonidos sintetizados (correcto, incorrecto, pasapalabra).
- Guardado local de nombre y leaderboard.

## Ejecutar local
1. Instala Node.js (v16+).
2. En la carpeta del proyecto ejecuta:
   npm install
   npm run dev
3. Abre http://localhost:5173

## Archivos importantes
- src/App.jsx -> componente principal
- src/assets/logo-spl.png -> logo (puedes reemplazarlo)
